# Protocolo RMP™ - Landing Page

Esta é a página de vendas criada com design Moderno e Futurista.

## Como usar
1. Basta abrir o arquivo `index.html` em seu navegador (Chrome, Firefox, Edge).
2. Para editar o vídeo:
   - Abra o `index.html` em um editor de texto (Bloco de Notas, VS Code).
   - Procure por `<div class="vsl-placeholder">`.
   - Substitua essa div pelo código de incorporação do YouTube/Vimeo/Wistia.

## Personalização
- **Cores**: Todas as cores estão definidas no topo do arquivo `style.css` na seção `:root`.
- **Imagens**: Se quiser adicionar fotos reais nos depoimentos, substitua `<div class="avatar-placeholder">` por `<img>`.

## Tecnologias
- HTML5 Semântico
- CSS3 (Variáveis, Flexbox, Grid, Animações)
- JavaScript (Intersection Observer para animações de scroll)
